Vypl

A Python REPL with Vim keybinds.

Vypl is an interactive Python shell designed for people who spend most of their time in Vim. It combines Python development with familiar Vim-style editing, navigation, and command workflows inside the REPL.

The goal is simple: provide a Python prompt that feels natural to Vim users. Instead of switching mental models when entering a REPL, you keep the same motions, commands, and editing habits you already use every day.

Install

pip install vypl

Or install from source:

git clone https://github.com/HoraDomu/Vypl
cd Vypl
pip install -e .

Run

vypl

Features

• Vim-style modal editing
• Normal and insert modes
• Vim motions and operators
• Named registers
• Ex commands
• Syntax highlighting
• Python autocompletion
